# Andoro Invoice Route Desk

A phone-friendly invoice and route planning app for Andoro & Sons.

## What It Does

- Scans daily invoice photos with browser-based OCR.
- Extracts invoice details such as customer, category, address, invoice number, amount, and dates.
- Tracks invoice status and category totals.
- Builds route stops from invoice addresses.
- Finds coordinates and optimizes the daily route.
- Opens the ordered route in Google Maps.

## Run Locally

Open `index.html` in a browser, or serve the folder with any simple static web server.

## Publish

This app can be published with GitHub Pages, Netlify, or Vercel because it only uses static files.
